MurOS apt repository

This host serves the signed Debian package repository for MurOS, the open source firewall. It is meant to be consumed by apt, not browsed by hand.

Quick install

On a clean Debian 13 (Trixie) machine, as root:

curl -fsSL https://apt.muros.org/install.sh | sudo bash

The script imports the signing key, registers this repository and installs the muros package. Then point a browser at https://<firewall-ip>.

Manual setup

If you prefer to add the repository yourself:

curl -fsSL https://apt.muros.org/muros.asc | sudo gpg --dearmor -o /usr/share/keyrings/muros-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/muros-archive-keyring.gpg] https://apt.muros.org stable main" | sudo tee /etc/apt/sources.list.d/muros.list
sudo apt update
sudo apt install muros

Later upgrades flow through apt: apt update && apt install --only-upgrade muros.

What is served here

  • install.shOne-line installer
  • uninstall.shClean removal

  • dists/Release metadata
  • pool/Package files (.deb)

Notes

  • The repository keeps the latest release. A plain .deb is also attached to every GitHub release.
  • Installation requires outbound HTTPS to apt.muros.org.
  • Documentation: quickstart and install guide.
